# Block Clipper - 测试文档

本文档包含完整的测试清单，用于验证 Block Clipper 扩展的所有功能。

## 🚀 安装测试

### 在 Chrome 中加载扩展

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目的 `.output/chrome-mv3` 目录
5. 确认扩展成功加载，没有错误

**预期结果**:
- ✅ 扩展图标显示在工具栏
- ✅ 无错误信息
- ✅ 显示欢迎通知

## 📝 剪藏功能测试

### 测试 1: 基本文本剪藏
**步骤**:
1. 访问任意网页
2. 选择一段纯文本
3. 按 `Ctrl+Shift+Y` (Mac: `Cmd+Shift+Y`)

**预期结果**:
- ✅ 显示通知"✓ Content clipped!"
- ✅ Options 页面和 Side Panel 可以看到剪藏的内容
- ✅ Markdown 格式正确

## 📊 已知限制

1. **Side Panel 关闭**: Chrome API 限制，快捷键无法关闭 Side Panel，需手动点击 X 按钮
2. **Firefox 支持**: 当前仅支持 Chrome（使用 Chrome Side Panel API）
3. **离线剪藏**: 必须在有网络连接的页面使用（因为依赖 content script）
