# 🚀 快速测试指南 - Block Clipper扩展

扩展已构建完成！按照以下步骤进行快速测试：

## 步骤1：加载扩展到Chrome

1. 打开Chrome浏览器
2. 访问 `chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择文件夹：`.output/chrome-mv3`
6. 确认扩展已加载（会显示Block Clipper图标）

## 步骤2：基础功能测试（5分钟）

### 测试1：基本剪藏功能
1. 打开任意网页（比如 https://example.com）
2. 用鼠标选中一些文字
3. 按键盘快捷键：`Ctrl+Shift+Y`
4. ✅ 应该看到通知："✓ Content clipped!"
5. 打开侧边栏（`Ctrl+Shift+S`）
6. ✅ 应该看到刚才剪藏的内容

### 测试2：侧边栏查看
1. 在侧边栏中点击刚才保存的内容
2. ✅ 应该看到完整的内容详情
3. ✅ Markdown格式应该正确显示
4. 点击"返回"按钮
5. ✅ 应该回到列表视图

### 测试3：上下文菜单
1. 在网页上选中一些文字
2. 右键点击
3. ✅ 菜单中应该有"Clip Selection"选项
4. 点击它
5. ✅ 应该看到成功通知

### 测试4：空选择处理
1. 不要选中任何文字
2. 按 `Ctrl+Shift+Y`
3. ✅ 应该看到错误通知："No content selected"

### 测试5：搜索功能
1. 剪藏几段不同的内容
2. 在侧边栏搜索框中输入关键词
3. ✅ 应该看到搜索结果更新

## 步骤3：在真实网站上测试

## 常见问题排查

### 问题1：侧边栏打不开
**原因：** Chrome Side Panel API限制
**解决：** 使用快捷键 `Ctrl+Shift+S` 或点击扩展图标打开

### 问题2：快捷键不工作
**检查：**
- 是否真的选中了文字？
- 快捷键是否被其他扩展占用？
- 访问 `chrome://extensions/shortcuts` 确认快捷键设置

### 问题3：通知不显示
**检查：**
- 浏览器通知权限是否开启
- 检查 `chrome://extensions/` 确认扩展有"通知"权限
